An application in web2py to compare two databases and generate a .sql file to fix delta
Steps to install
--Download the app and rename it to "comparedb". 
--Kindly paste the folder in your web2py applications folder.
--Start the web2py server and visit "http://127.0.0.1:8000/comparedb/default/enter".
--Please enter the Name, Username and password for both the databases.
--The resulting page will show the Schema difference between the two database.
--An output.sql file will be generated in the web2py directory which when run in production environment will generate the delta.
